# Proyecto SALT 

### Información del repositorio:
| branch | descripción |
|--------|-------------|
| [firmware](https://github.com/martinanus/salt/tree/firmware) |  se encuentra el proyecto completo de STM32CubeIDE para cargar en la placa STM32 Nucleo-144 |
| [mqtt](https://github.com/martinanus/salt/tree/mqtt) | se encuentra el código utilizado para simular la central operativa |
| [esp32](https://github.com/martinanus/salt/tree/esp32) | se encuentra el archivo esp32_salt.ino para configurar el módulo ESP32-devkit-C incluido en el hardware |
| [hardware_principal](https://github.com/martinanus/salt/tree/hardware_principal) | se encuentra el proyecto en Kicad 6.0 de la placa principal del sistema |
| [hardware_ihm](https://github.com/martinanus/salt/tree/hardware_ihm) | se encuentra el proyecto en Kicad 6.0 de la placa interfaz humano-máquina del sistema |
